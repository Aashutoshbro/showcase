## Getting Started

```
git clone https://github.com/sumitbro/e_consult_4rth-sem.git

cd e_consult_4rth

flutter pub get 

flutter packages pub run build_runner build --delete-conflicting-outputs

flutter test
 
flutter run lib/main.dart

```
